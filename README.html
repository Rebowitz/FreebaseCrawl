Command Line Freebase Search Sample Instructions

A starter application has been generated for you.

  1. Install Java Platform (JDK 6 or higher).
  2. Install Maven (3.1.0 or higher). You may need to set your JAVA_HOME.
  3. Download the starter application and unzip it.
  4. Open a terminal, change to the freebase-cmd-line-sample directory you just unzipped, and execute: 
    mvn compile
mvn -q exec:java
    This will compile and run your starter application in the command line.

Congratulations, you are up and running! At this point you will want to add calls into the Freebase Search to the /src/main/java/com/google/api/services/samples/freebase/cmdline/FreebaseSample.java file. Please read the file carefully, it contains detailed information in the comments.

To enable logging of HTTP requests and responses (highly recommended when debugging), please take a look at the logging.properties file.

For more information on the Freebase Search Java library surface you can visit:

https://develpers.google.com/resources/api-libraries/documentation/freebase/v1/java/latest/

Also check out the Java Client Library documentation, and get more information on the Freebase Search at:

https://developers.google.com/freebase/

Setup Project in Eclipse

Prerequisites: install Eclipse, and the Maven plugin.

  • Setup Eclipse Preferences
    • Window > Preferences... (or on Mac, Eclipse > Preferences...)
    • Select Maven
      • check on "Download Artifact Sources"
      • check on "Download Artifact JavaDoc"
  • Import freebase-cmd-line-sample project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find freebase-cmdline-sample and click "Next"
    • Click "Finish"
  • Run
    • Right-click on project freebase-cmd-line-sample
    • Run As > Java Application
    • If asked, type "FreebaseSample" and click OK
